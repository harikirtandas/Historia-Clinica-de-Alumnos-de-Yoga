Desarrollé una app web interna para un estudio de yoga que registra y gestiona historias clínicas de alumnos antes de iniciar clases. Está construida con Python + Flask, usa SQLite, y permite CRUD completo, búsqueda por alumno, y exportación de PDF (individual y listado). La UI es simple y clara para uso diario en recepción, con datos sensibles protegidos para uso local. Ideal para organizar la información médica relevante y agilizar la entrevista inicial.
